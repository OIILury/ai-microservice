ℹ️ Comment fonctionne cette application ?

▼

Architecture "Local-First"

Ce microservice utilise Ollama en local pour garantir une confidentialité totale. Le pipeline de correction exécute deux passes : une pour la grammaire et une pour le style (reformulation).

La télémétrie est stockée localement dans une base SQLite pour analyser les performances sur Jetson Orin NX.
